feat: 添加CSS选择器全屏功能

## 功能描述
支持在全屏时只显示iframe中指定CSS选择器的元素，适用于视频播放器、文章阅读等场景。

## 主要变更

### 新增功能

- 🔄 全屏时自动应用选择器
- ↩️ 退出全屏时自动恢复原始样式

## 技术实现

### Electron版本
通过IPC通信在iframe中执行JavaScript代码，直接操作DOM元素。

### Chrome扩展版本
通过postMessage与content script通信，再由background script在iframe中执行脚本。

## 使用示例

```javascript
// 配置网站
{
  title: "YouTube",
  url: "https://youtube.com",
  targetSelector: "#movie_player"  // 新增字段
}

// 全屏时只显示播放器
```

## 常用选择器
- YouTube: #movie_player
- Bilibili: .bilibili-player
- 知乎文章: .Post-Main
- 通用视频: video
